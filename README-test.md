## ūüĒ¨ What does GeneViz do?

GeneViz is a lightweight yet powerful Python script that generates **publication‚ÄĎquality pairwise microsynteny plots** for any two genomic regions, within the same species or across different species. It combines:

- **BLAST‚ÄĎbased homology** ribbons (colored by bitscore)
- **Gene structure** (exons/introns with direction arrows)
- **Transposable element (TE)** tracks with motif labels
- **Automatic orientation detection** (reverse‚ÄĎcomplement when needed)
- **Core region highlighting** (e.g., conserved domains)

All you need are GFF annotation files, a genome FASTA, and BLAST output (optional ‚Äď the script can run BLAST for you). GeneViz produces SVG and PDF vector figures ready for publication.

## ‚ú® Key Features

- **Single‚ÄĎspecies or cross‚ÄĎspecies** ‚Äď compare genes on the same genome or between two different genomes.
- **Automatic reverse complement** ‚Äď detects if the second region should be flipped based on BLAST hit directions.
- **TE track** ‚Äď draws transposable elements above/below the chromosome, extracts motif names from GFF attributes.
- **Gene structure** ‚Äď shows exons (colored boxes) and introns (lines), with an arrow indicating transcription direction.
- **Core range highlighting** ‚Äď mark a specific interval (e.g., protein domain) in red within genes.
- **BLAST ribbon coloring** ‚Äď polygons connecting homologous regions, colored by bitscore (gray ‚Üí purple ‚Üí red).
- **Vector output** ‚Äď SVG and PDF (via `cairosvg`) for easy editing.
- **Highly customizable** ‚Äď dozens of command‚ÄĎline options to control sizes, colors, gaps, etc.
- **No complex installation** ‚Äď just a single Python script; all Python dependencies are listed in `requirements.txt`.
